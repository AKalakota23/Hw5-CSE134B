My Projects

Smart Watch | Fall 2024
  • Designed and built Bluetooth-enabled smart watch using an ESP32 microcontroller.
  • Integrated OLED display for real-time notifications from iPhone via BLE communication.
  • Configured Apple Notification Center Service (ANCS) on ESP32.
  • Utilized U8g2 library to customize OLED display output.
Rodent Recon | Fall 2024
  • Built a security system using Raspberry Pi Zero 2 W and camera v2.
  • Trained machine learning models to detect squirrels in backyard.
  • Developed automated mechanism to trigger high-pitched noise when squirrels are detected.
  • Engineered wireless communication system between camera, sensor, receiver, and speaker.
IEEE HardHack Hackathon | Spring 2024
  • Developed Arduino project to enhance deaf people's situational awareness during nighttime navigation.
  • Utilized ultrasonic sensor and camera to record video when people approach.
  • Implemented SpeechRecognition to transcribe audio to text and display on LCD.
  • Performed sentiment analysis and lit LEDs based on connotation.
Reinforcement Learning for Self-Driving Car | Summer 2023
  • Designed and developed autonomous robot using Soft-Actor-Critic (SAC) algorithm.
  • Employed distance to track centerline as reward signal for training.
  • Utilized Virtual Reality for real-time position and orientation tracking.
  • Presented findings at Texas A&M Summer Research Symposium.
